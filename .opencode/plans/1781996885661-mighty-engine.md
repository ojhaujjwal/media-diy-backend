# Plan: Remove `tryPromise` — Manual SigV4 Presigned URL Signing

## Summary

Replace `@aws-sdk/client-s3` + `@aws-sdk/s3-request-presigner` (`Effect.tryPromise` + `getSignedUrl`) with a manual AWS SigV4 presigned URL implementation using Web Crypto API (`crypto.subtle`).

## Files to Create

### `src/infrastructure/persistence/sigv4.ts`

New module implementing AWS Signature Version 4 presigned URL generation.

**Inputs:**
- `accessKeyId: string`
- `secretAccessKey: string`
- `region: string` (for R2: `"auto"`)
- `endpoint: string` (e.g., `https://{account}.r2.cloudflarestorage.com`)
- `bucket: string`
- `key: string`
- `contentType: string`
- `expiresIn: number` (seconds)

**Output:** `Effect.Effect<string, MediaContentsRepositoryError, never>` — the full presigned URL.

**Algorithm:**

1. Derive `amzDate` (`YYYYMMDDTHHMMSSZ`) and `dateStamp` (`YYYYMMDD`) from current time (UTC)
2. Build the credential scope: `{dateStamp}/{region}/s3/aws4_request`
3. Build signed headers (alphabetical): `content-type;host`
4. Build canonical query string: `X-Amz-Algorithm=AWS4-HMAC-SHA256&X-Amz-Credential={encoded_cred}&X-Amz-Date={amzDate}&X-Amz-Expires={expires}&X-Amz-SignedHeaders={encoded_signed_headers}`
5. Build canonical request:
   ```
   PUT
   /{bucket}/{key}
   {canonical_query}
   content-type:{contentType}
   host:{host}
   
   content-type;host
   UNSIGNED-PAYLOAD
   ```
6. Build string to sign:
   ```
   AWS4-HMAC-SHA256
   {amzDate}
   {dateStamp}/{region}/s3/aws4_request
   {sha256(canonical_request)}
   ```
7. Compute signing key via HMAC chain: `AWS4{secret}` → `kDate` → `kRegion` → `kService` → `kSigning`
8. Compute signature = `hex(hmac(kSigning, string_to_sign))`
9. Construct final URL: `{endpoint}/{bucket}/{key}?{canonical_query}&X-Amz-Signature={signature}`

**Web Crypto API usage:**
- `crypto.subtle.importKey("raw", ..., {name:"HMAC", hash:"SHA-256"}, false, ["sign"])` — import signing key
- `crypto.subtle.sign("HMAC", key, data)` — HMAC-SHA256
- `crypto.subtle.digest("SHA-256", data)` — SHA256 hash
- All wrapped in `Effect.tryPromise` (this is for platform API bridging, not the AWS SDK pattern being removed)

**Error handling:** Wrap all crypto operations in `Effect.tryPromise` and map errors to `MediaContentsRepositoryError`.
